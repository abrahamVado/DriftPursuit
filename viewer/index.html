    All controls are mirrored via keyboard shortcuts. Buttons update in real time as modes change.

    Quickstart

    1. Select an aircraft model to load its assets.
    2. Watch the HUD for “ready”, then toggle Manual Control or use keyboard shortcuts.
    3. Use the Tracked Aircraft dropdown (or press ] / [) to switch the camera between active planes.
    4. Use WASD/RF for translation, QE for yaw, and arrow keys for pitch/roll.
    5. Cycle the autopilot route to see preset waypoint loops.
